# Vanya Handlooms - Premium Heritage Collection

A premium e-commerce experience for Vanya Handlooms, showcasing timeless heritage weaves including Handloom, God Clothes, and Fancy Articles.

## Features
- **Modern Minimalist Design**: High-end aesthetics with smooth animations.
- **Automated Checkout**: Orders are sent directly to the owner via WhatsApp.
- **Responsive Layout**: Optimized for all devices.
- **Specialized Categories**: 
  - **Handloom**: Traditional hand-woven masterpieces.
  - **God Clothes**: Divine attire for sacred occasions.
  - **Fancy Articles**: Elegant, high-fashion pieces.

## Deployment
This project is designed to be deployed as a static site on Vercel or GitHub Pages.

### How to Order
When a user places an order, the system automatically captures their details and sends a formatted WhatsApp message to the business owner for processing.
